# LumenEX

**LumenEX** is a standalone C lexer library and development project maintained by **[AsterIDE](https://asteride.dev)**. It is designed to tokenize source text for multiple languages through a stable public API built around immutable source ranges and reusable token structures.

---

>[!IMPORTANT]
> **LumenEX** is still under active development. Additional language support and API refinements are planned, so please expect changes as the project evolves.

>[!NOTE]
> **LumenEX** was created with the needs of **[AsterIDE](https://github.com/Aster-IDE/AsterIDE)** in mind, but it is intended to be a general-purpose library that can be used beyond any single editor or IDE.
